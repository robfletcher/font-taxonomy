The Geometry of (Web) Type

This is a collection of some of my favorite free web fonts classified and displayed in homage to Stephen Coles' fantastic book The Geometry of Type (published as The Anatomy of Type in the US).

I have selected fonts that I like personally, are popular, are resonable stand-ins for mainstream fonts or otherwise seem useful (such as fonts with a variety of styles).

Disclaimer: I am not a typography expert just an interested amateur so glaring errors in classification are to be expected! This page is a GitHub project so if you feel inclined to correct anything then please send me a pull request.
